# About : 
**_Dams3D_** is my project for a custom 3D renderer developed using OpenGL, C++ and GLSL.
It's a WIP but I would love to expand it to a custom 3D game engine in the future.

Main sources used for this project are https://learnopengl.com/ website by Joey de Vries, and Victor Gordan youtube series on OpenGL.

# Current Features :
- Functional diffuse / specular maps.
- 3D Model Loading ( currently for gltf file format only ).
- Lightning : Blinn–Phong implementation &
directional/point/spotlight lightmodes.
- Camera : moving in real-time via user inputs & orthographic/perpective implementation.

# Next Features :
- Expand 3D Model Loading using Assimp.
- Handle Transparency.
- Add Post-Processing.
- Add Shadow Maps
- Implement Shell Texturing for fur / moss / hair representation.
